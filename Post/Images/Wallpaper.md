Mi configuracion
Usar nitrogen
Instalar
```
pacman -S nitrogen
```
Hacer un archivo de autoarranque
``` micro ~/.config/qtile/autostart.sh``` y escribe ```nitrogen --restore &``` sal de editar, ejecuta ```chmod +x ~/.config/qtile/autostart.sh```

Hacer que el archivo de autoarranque lo ejecute qtile
